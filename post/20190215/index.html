一种用于预测GitHub中流行度的交叉存储库模型

原文标题： A Cross-Repository Model for Predicting Popularity in GitHub

地址： http://arxiv.org/abs/1902.05216

作者： Neda Hajiakhoond Bidoki, Gita Sukthankar, Heather Keathley, Ivan Garibay

摘要： 社交编码平台，如GitHub，可以作为自然实验室，通过跟踪程序员采用的代码模式来研究创新的传播。本文重点讨论了预测软件存储库随时间流行的问题;我们的目标是预测人气相关事件的时间序列（分支和观察）。特别是，我们对跨存储库模式感兴趣 - 一个存储库上的事件如何影响其他存储库？我们提出的LSTM（长短期记忆）递归神经网络在多个活动存储库中集成事件，优于基于单个存储库的标准ARIMA（自回归集成移动平均）时间序列预测。 LSTM利用跨存储库信息的能力使其比标准时间序列预测具有显著优势。

从信息流和人才流动推断区域经济状态

原文标题： Regional economic status inference from information flow and talent mobility

地址： http://arxiv.org/abs/1902.05218

作者： Jun Wang, Jian Gao, Jin-Hu Liu, Dan Yang, Tao Zhou

摘要： 新的数据已被用于及时估计社会经济状况，然而，直接比较社会关系和人才流动的使用仍然很少。在这封信中，我们根据两个网络的结构特征估算了区域经济状况。一个是建立在社交媒体上的以下关系的在线信息流网络，另一个是基于具有高等教育的求职者的匿名简历数据构建的离线人才移动网络。我们发现，虽然两个网络的结构特征与经济地位相关，但相对较小规模的人才流动网络对国内生产总值（GDP）具有更强的预测能力。特别是，结构特征的综合指数可以解释高达约84％的GDP差异。结果表明，未来的社会经济研究将更加关注具有成本效益的人才流动数据。

应用于GitHub的网络语义分割

原文标题： Network Semantic Segmentation with Application to GitHub

地址： http://arxiv.org/abs/1902.05220

作者： Neda Hajiakhoond Bidoki, Gita Sukthankar

摘要： 在本文中，我们介绍了社会网络分析的网络语义分割的概念。我们认为GitHub社交编码网络一直是研究人员和软件开发人员关注的焦点。网络语义分段描述了将每个用户与诸如感兴趣的主题之类的类标签相关联的过程。我们使用网络重要连接来扩充节点属性，然后使用机器学习方法来聚集用户。我们将结果与使用社区检测算法执行的网络分段和通过使用节点属性进行群集执行的网络分段进行比较。将结果与语义段中的社区多样性以及主题进行比较

专利质量的事前衡量揭示了引用网络增长的内在适应性

原文标题： Ex-ante measure of patent quality reveals intrinsic fitness for citation-network growth

地址： http://arxiv.org/abs/1902.05265

作者： K. W. Higham, M. Governale, A. B. Jaffe, U. Zülicke

摘要： 我们已经从授予专利时已知的相关发明的属性构建了适应性参数，其表征了要引用的专利的内在吸引力。这种外生获得的适应性被证明可以确定引文网络的时间增长以及优先附着和过时引起的老化机制，其不参考个别专利的特征而进行操作。我们的研究打开了一个窗口，可以定量地理解丰富的富裕和更富裕的范式之间的相互作用，这些范式被建议用来管理现实世界复杂网络的增长动态。
