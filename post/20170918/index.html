问答网站的活动

地址： http://arxiv.org/abs/1709.05111

作者： Tiago Santos, Simon Walk, Roman Kern, Markus Strohmaier, Denis Helic

摘要： 互联网上的数百万用户讨论了各种问题和答案（Q＆A）实例。然而，并不是所有的实例和主题都受到同样的关注，因为一些繁荣和实现自我维持的活动水平，而另一些则不能吸引用户，或者永远不会成长为超越小的利基社区或变得不活跃。因此，不仅要更好地理解，而且必须提炼定义和管理可持续问答实践的决定因素和规则。我们的目标是为社区管理者提供量化方法，使他们更好地了解，控制和培养他们的社区，从而有助于使Web成为更有效的交换信息的地方。为此，我们从StackExchange网络的50个随机选择的Q＆A实例中提取，建模和集群基于用户活动的时间序列，以表征用户行为。我们发现四种不同类型的用户活动时间模式，主要根据用户的活动频率而有所不同。最后，通过以前确定的用户活动简介，我们将50个问答实例的总体活动分解为三个不同的活动配置文件。我们对问答实例的分类与底层社区的发展阶段和成熟度相适应，这可能有助于这些实例的运营商不仅可以定量评估状态和进度，而且还可以优化社区建设工作。

最优通货膨胀目标：基于主体模型的见解

地址： http://arxiv.org/abs/1709.05117

作者： Jean-Philippe Bouchaud, Stanislao Gualdi, Marco Tarzia, Francesco Zamponi

摘要： 中央银行应针对哪个通货膨胀水平？我们在简化的经济代理模型的背景下调查这个问题。根据描述代理人微观行为（特别是通货膨胀预期）的参数的价值，我们发现宏观层面上出现了令人惊讶的多种行为。没有任何货币政策，我们的反倾销经济可能处于高通货膨胀/高产出状态，或处于低通货膨胀/低产出状态。过度通货膨胀，滞胀，通货紧缩和商业周期也是可能的。然后，我们介绍一个中央银行的泰勒规则通货膨胀目标，并研究产生的总体变量。我们的主要结果是通货膨胀目标太低通常不利于控制CB的经济。一个症状是通货膨胀的持续不足，也许与目前的宏观经济形势相似。更高的通货膨胀目标可以缓解这一困境，这些目标被认为可以改善失业率和负利率事件，直到储蓄的下降成为不可接受的程度。我们的结果与标准DSGE模型的预测形成对照。

关于通过矩阵函数定义的网络索引的稳定性

地址： http://arxiv.org/abs/1709.05132

作者： Stefano Pozza, Francesco Tudisco

摘要： 识别网络中的重要组件是网络分析的主要目标之一。根据矩阵$ f（A）$的函数的适当条目来定义节点或一组节点的重要性的流行和有效的度量。这些措施特别相关，因为它们能够捕获涉及节点的连接的全局结构。然而，计算$ f（A）$的条目需要大量的计算量。在这项工作中，我们讨论了关于边结构变化估计$ f（A）$条目变化的问题。直觉表明，如果新图$ \ tilde G $中的连接拓扑没有显着扭曲，$ G $中的相关组件将在$ \ tilde G $中维持其主角。我们提出了几个界限，给出这种直觉的数学推理，特别表明，条目$ f（A）_ {k \ ell} $的变化幅度以$ G $中的最短路径距离指数地衰减，从被扰动的边触摸的节点组中分离$ k $或$ \ ell $。此外，我们提出一种简单的方法，利用$ f（A）$的计算来同时计算$ G $的全对最短路径距离，基本上没有额外的成本。由于边连接趋向于更频繁地或经常受到噪声影响的节点，在图中具有微小的作用，并且与最中心的节点相距甚远，所以提出的界限是特别相关的。

通过爆发加强的机制建模相关的爆发

地址： http://arxiv.org/abs/1709.05150

作者： Hang-Hyun Jo

摘要： 时间序列或事件序列在自然和社会现象中的时态相关性的特征在于幂律衰减自相关函数与衰减指数$ \γ$。这样的时间相关性可以通过幂指数分布的异常时间与指数$ $α$和/或内部时间之间的相关性来理解。后者通常称为相关爆发，最近通过测量幂指数分布式突发火车以指数$ \ beta $进行了研究。对于不相关的内部时间，已经建立了$ \ alpha $和$ \ gamma $之间的尺度关系，而对相关的内部时间对时间相关性的影响知之甚少。为了研究这些影响，我们设计了用于相关脉冲串的突发爆发模型，通过该模型，可以调整内部时间之间的相关程度，同时保持相同的时间分布。我们在数值上发现，在不相关的情况下，异常时间之间的足够强的相关性可能违反$ \ alpha $和$ \ gamma $之间的尺度关系。 $ \ alpha $对$ \ beta $的非平凡依赖也可以在某些范围的$ \ alpha $中找到。我们的结果的含义就是在各种时间尺度上对突发火车的等级组织进行了讨论。

系统文献评论中的众包论文筛选

地址： http://arxiv.org/abs/1709.05168

作者： Evgeny Krivosheev, Fabio Casati, Valentina Caforio, Boualem Benatallah

摘要： 文学评论使科学家们能够站在巨人的肩膀上，展现出有前途的方向，总结进展，并指出现有的研究挑战。同时进行系统的文献综述是一个艰巨而又昂贵的过程。在过去十年中，有关文献综述众包的研究。本文探讨了众包的可行性，以便在结果，时间和精力方面促进文献审查过程，以及根据可用预算来确定哪些群众战略提供最好的结果。特别是我们专注于文献审查过程的筛选阶段，我们贡献和评估确定测试大小，每篇文章所需标签和分类功能的方法，以及分阶段分解众包流程以改进结果的方法。最后，我们根据在Crowdflower上运行的实验介绍我们的发现。

通过复杂网络聚集音乐片段：评估吉他独奏

地址： http://arxiv.org/abs/1709.05193

作者： Stefano Ferretti

摘要： 音乐作品可以被模仿为复杂的网络。这促进了创新的方式来分类音乐，为多媒体领域的新颖应用铺路，如音乐教学，多媒体娱乐和数字音乐生成。通过其主要指标聚合这些网络，可以对相似的音乐曲目进行分组。为了显示该方法的可行性，我们在吉他独奏数据集上提供了结果。

连续时间信息级联活动整形的谱方法

地址： http://arxiv.org/abs/1709.05231

作者： Kevin Scaman, Argyris Kalogeratos, Luca Corinzia, Nicolas Vayatis

摘要： 信息级联模型捕获社会网络中用户活动的动态属性。在这项工作中，我们为连续时间信息级联模型下的活动整形开发了一个新颖的框架，通过分配可以改变流程传播的目标资源，使管理员能够进行本地控制。我们的框架使用了危害矩阵的谱半径优化，这个数量已被证明可以驱使网络中的最大影响，同时在用于最小化级联的影响的同时享受简单的凸松弛。此外，还讨论了诸如检疫和节点免疫的用例，以突出提出的活动整形框架的一般性。最后，我们介绍了NetShape影响最小化方法，通过对真实社会网络的模拟，将其与基线和最先进的方法相比较。

在线回声室的空间维度

地址： http://arxiv.org/abs/1709.05233

作者： Marco T. Bastos, Dan Mercea, Andrea Baronchelli

摘要： 本研究探讨了Brexit全民投票活动期间Twitter上回声室通信的地理依赖关系。我们回顾关于滤波器气泡，回波室和极化的文献，以测试五个假设，假设回波室通信与物理世界中的同步相关，主要是倡导运动双方的用户之间的地理接近度。结果支持了假期运动中的回音室与地理特征相关的假设，而在剩余运动中则发现了反向关系。这项研究提供的证据表明，地理上接近的社会地域与观察到回波室通信的极化政治讨论相互作用。文章最后讨论了这些发现和对过滤泡和回声室研究的贡献。
