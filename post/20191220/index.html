沟通社区与节点表示：利用社区发现的图嵌入

原文标题： Bridging the Gap between Community and Node Representations: Graph Embedding via Community Detection

地址： http://arxiv.org/abs/1912.08808

作者： Artem Lutov, Dingqi Yang, Philippe Cudré-Mauroux

摘要： 图嵌入已成为许多数据挖掘和分析系统的重要组成部分。当前图嵌入靠近任采样来自的曲线图的大量节点对经由随机优化学习节点的嵌入或通过计算上昂贵的矩阵分解技术因式分解的曲线图的高次接近/邻接矩阵。这些方法通常需要显著资源，为学习的过程，依赖于多个参数，这限制了其应用在实践中。此外，大多数现有的图嵌入技术的在一个特定的度量空间有效地操作只（例如，与余弦相似性产生的一个），不保留输入图的高阶结构特征和不能自动确定嵌入尺寸的有意义的数字。典型地，所产生的嵌入不容易解释的，这复杂化的进一步分析和限制了其适用性。为了解决这些问题，我们提出了DAOR，高效，无参数图嵌入技术生产度量空间，坚固，紧凑的和可解释的嵌入无需任何手动调节。相比于一个国家的最先进的打图嵌入算法，DAOR产生在两个节点分类结果有竞争力（其好处形成高阶接近）和链路预测（其依赖于低阶接近大部分）。与现有的技术，但是，DAOR不需要任何参数调整和几个数量级提高的嵌入生成速度。我们的方法有因此野心极大地简化并加快涉及图表示学习数据分析任务。

摘要： 网络嵌入的目的是在网络中的节点转换到低维嵌入载体。最近，异构网络已证明是有效的表示数据的各种信息。然而，异构网络从嵌入的不平衡问题遭受的，即关系类型（或网络中的边的关于类型的号码）的大小是不平衡的。在本文中，我们设计了新的异构网络中嵌入的方法，称为BHIN2vec，它认为网络中的所有关系类型之间的平衡。我们认为，在异构网络中嵌入的同时解决多个任务，每个任务对应于网络中的每个关系类型。分开后跳过克损失成对应于不同的任务多重损失，我们提出了一个新的随机行走策略，通过考虑相关培训的比例集中于具有高损耗值的任务。不同于以往的随机游走战略，我们提出的随机行走策略根据不同任务之间的相关培训率，这导致了节点嵌入一个平衡的训练生成训练样本。我们对节点分类和推荐大量的实验证明了相比于国家的最先进的方法BHIN2vec的优越性。此外，基于相对锻炼比例，我们分析每一个关系型的多少在嵌入空间中表示。

摘要： 该阿克塞尔罗德模型，因为它的社会影响力和文化传播的建议得到了广泛的研究。特别是，统计物理的社会集中在相变作为其两个主要参数， F 和 Q 的函数的存在。这些参数发挥设置系统，这意味着相变的关键特性在很大程度上取决于初始状态的初始条件了关键作用。在这项工作中，我们表明，阿克塞尔罗德模型的 F极限进行第二相变 RIGHTARROW infty 上一个完整的图。这个过渡是等效于一个Erd “OS-R ‘时它是在相互作用的可能性方面，在初始状态中所述在随机网络恩义相变。我们还发现，该参数通过起着稀疏拓扑关键作用倒塌的参数 F 的不同值的过渡曲线，我们探讨这个崩溃，导致这种动力学机制的程度。
