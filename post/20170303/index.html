秘鲁共和国议会立法共同主办网络中的时间依赖性社区结构

地址： http://arxiv.org/abs/1510.01002

作者： Sang Hoon Lee, José Manuel Magallanes, Mason A. Porter

摘要： 我们研究在秘鲁国会的时间依赖立法共同主办网络的社区结构，我们简要比较它们在美国参议院的立法联合赞助网络。为了研究这些立法，我们采用时间网络的多层表示，其中每层中的立法者彼此连接，其权重基于它们共同发起的票据数量。然后，我们使用多层模块性最大化来检测这些网络中的社区。从我们的计算，我们能够捕获在2006 - 2011年秘鲁国会的权力变化。例如，我们观察到“机会主义者”的出现，他们从一个社区转向另一个社区，以及具有凝聚力的立法团体，其初始组成立法者从不改变社区。有趣的是，许多机会主义者属于在国会中赢得多数的集团。

足球比赛中球员的方向对准和追逐的统计特性

地址： http://arxiv.org/abs/1611.03880

作者： Takuma Narizuka, Yoshihiro Yamazaki

摘要： 着眼于足球游戏中两个相互作用的运动员的运动，一个运动员和最近的对手运动员的两个速度矢量表现出强对准。特别是，我们发现存在一个特征人际距离$ r \ simeq 500 $ cm低于其对齐的圆形方差迅速减少。通过引入订单参数$ \ phi（t）$来测量玩家的速度向量的对齐程度，我们还发现最近玩家的速度向量之间的角度分布变为包裹Cauchy（$ \ phi \ lesssim 0.7 $ ）和von Mises和包裹的Cauchy分布的混合（$ \ phi \ gtrsim 0.7 $）。为了理解这些发现，我们构建一个简单的模型为两个相互作用的球员的运动与以下规则：追逐球员和重置的追逐。我们数字表明，我们的模型成功地重现了从实际数据获得的结果。此外，从数值研究，我们发现有另一个特征距离$ r \ simeq 1000 $ cm低于玩家的追逐开始。

指导点过程的变分政策

地址： http://arxiv.org/abs/1701.08585

作者： Yichen Wang, Grady Williams, Evangelos Theodorou, Le Song

摘要： 时间点过程是对事件发生进行建模的有力工具，并且在社会科学中具有大量的应用。在本文中，我们考虑如何设计点过程的最优控制策略的问题，使得由过程驱动的随机系统被引导到目标状态。特别地，我们利用关键洞察力从最佳度量和变分推理的角度观察随机最优控制问题。我们进一步提出一个凸优化框架和一个有效的算法来更新政策适应当前的系统状态。对合成和真实世界数据的实验表明，我们的算法可以比其他随机控制方法更精确和有效地引导用户活动。

审议增加了人群的智慧

地址： http://arxiv.org/abs/1703.00045

作者： Joaquin Navajas, Tamara Niella, Gerry Garbulsky, Bahador Bahrami, Mariano Sigman

摘要： 许多独立估计的聚合可以胜过最准确的个体判断。这个百年的发现，通常被称为“人群的智慧”，最近已经应用于从癌症的诊断到财务预测的问题。人们普遍认为，集体准确性的关键是保持个人在人群中的独立性。与这个普遍的观点相反，我们表明审议和讨论提高了集体智慧。我们问现场人群（N = 5180）回答一般知识问题（例如埃菲尔铁塔的高度）。参与者首先单独回答，然后审议并以五个小组作出协商一致的决定，最后提供了修订的个人估计。我们发现，共识和修订后的估计比独立意见的统一集合可以实现的偏差和多样性更小。因此，不同的共识决定的平均值比总结独立意见要准确得多。即使只有四个共识选择，也超过了成千上万个人的智慧。我们的研究结果表明，独立辩论的平均信息是一个非常有效的策略，利用我们的集体知识。

Erdos - Renyi随机图形拓扑随机吗？

地址： http://arxiv.org/abs/1703.00210

作者： Keith Smith, Javier Escudero

摘要： 不，我们证明Erdos - Renyi随机图不是拓扑随机的。这开始提供关于为什么随机图不能解释真实世界网络的拓扑特征的数学解释。这对网络科学中实现的随机化过程具有深远的影响。

使用积极的未标记学习和领域适应，从在线评论中识别产品召回的领先指标

地址： http://arxiv.org/abs/1703.00518

作者： Shreesh Kumara Bhat, Aron Culotta

摘要： 消费者保护机构负责保护公众免受危险产品的侵害，但其管辖下的数千种产品很难确定并快速响应消费者投诉。从消费者的角度来看，在线评论可以提供产品缺陷的证据，但是手动筛选数百个评论并不总是可行的。在本文中，我们建议一个系统来挖掘亚马逊的评论，以确定可能造成安全或健康危害的产品。由于这项任务的标签数据很少，我们的方法将积极的无标签学习与领域适应相结合，对来自美国消费品安全委员会的消费者投诉进行分类。对于手动注释亚马逊产品评论的验证集，我们发现我们的方法导致绝对F1得分改善比最佳竞争基线高8％。此外，我们将分类器应用于亚马逊对已知召回产品的评论;分类器在45％的产品召回之前识别报告安全危害的评论。这表明该系统可能能够提供预警系统，以在官方召回被宣布之前提醒消费者有害产品。

地球同步路由和寻址方案

地址： http://arxiv.org/abs/1703.00520

作者： Ivan Voitalov, Rodrigo Aldecoa, Lan Wang, Dmitri Krioukov

摘要： 在任何电信网络中，特别是在物联网（IoT）网络中路由的关键要求是可扩展性。路由必须在网络中的任何源和目的地之间路由分组，而不会招致随着网络大小和动态增加而快速增长的不可管理的路由开销。在这里我们提出一种寻址方案和耦合网络拓扑设计方案，保证基本上最优的路由可扩展性。 FIB大小尽可能小，等于节点具有的邻接数，而路由控制开销被最小化，因为即使在网络中的灾难性故障时也交换了近零路由控制消息。关键的新成分是寻址方案，其纯粹是局部的，仅基于节点的地理坐标和中心性测量，并且不需要用于网络嵌入的任何复杂的非局部计算或全局网络拓扑知识。为这些优势支付的价格是，网络拓扑不能是任意的，但应该遵循特定的设计，导致类似Internet的拓扑。所提出的方案可以最容易地部署在覆盖网络中，以及在其中可获得地理位置信息的其它网络部署中，并且其中网络拓扑可以遵循设计规范而增长。

人类方言的空间演化

地址： http://arxiv.org/abs/1703.00533

作者： James Burridge

摘要： 人类方言的地理模式是历史的结果。在这里，我们制定了语言变化的简单空间模型，表明这种历史进化的最终结果在某种程度上可以是可预测的。该模型显示，语言方言区域的边界由类似于表面张力的长度最小化效应控制，该表面张力由可以引起曲率的人口密度变化以及海岸线或类似边界的形状所介导。方言区域的可预测性出现，因为这些效应将驱动许多复杂的，随机化的早期状态朝向较小数量的稳定的最终配置之一。该模型能够再现观察和预言的辩证学家。这些包括方言连续性，同语言捆绑，扇动，城市方言特征的波浪式传播，以及人类活动对区域可以支持的方言数量的影响。该模型还提供了一个分析形式，用于给出地理和语言距离之间的关系，以及曲线的一般化以解释人口中心的存在。一个简单的修改允许我们分析表征“表观时间”的语言概念。

创意社区揭秘：美化统计概述

地址： http://arxiv.org/abs/1703.00800

作者： Nam Wook Kim

摘要： 在线社区正在改变创意专业人士如艺术家和设计师分享想法，接收反馈和寻找灵感的方式。虽然他们越来越受欢迎，但迄今为止几乎没有研究。在本文中，我们调查Behance，一个在线社区网站的广告素材，以保持与他人的关系，并展示他们的作品从平面设计，插图，摄影和时尚等各个领域。我们采取定量方法研究关于该网站的三个研究问题。什么吸引跟随和欣赏艺术品Behance？主题周围存在什么样的活动？最后，颜色是否在吸引欣赏方面发挥作用？总之，男性建议更多的追随者和赞赏，大多数用户关注几个主题，灰度颜色意味着更少的赞赏。这项工作作为一个创意社区的初步概述，以后的研究可以建立。

多社会网络中的友谊维护和预测

地址： http://arxiv.org/abs/1703.00857

作者： Roy Ka-Wei Lee, Ee-Peng Lim

摘要： 由于在线社交网络（OSN）的激增，用户发现自己参与多个OSN。这些用户保留活动跟踪，因为他们保持友谊和与这些OSN中的其他用户交互。在这项工作中，我们分析用户如何保持在多个OSN的友谊，研究在Twitter和Instagram上有帐户的用户。具体来说，我们研究用户的友谊与多个OSN中友谊分布的均匀性的相似性。我们的研究表明，大多数用户在Twitter和Instagram喜欢在两个OSN中保持不同的友谊，保持只有一小群共同的朋友在整个OSN。基于我们的实证研究，我们进行链接预测实验，以使用邻域特征，邻域友好维护特征和交叉链接特征来预测多个OSN中的缺失友谊链接。我们的链路预测实验表明，非监督方法可以在使用另一OSN数据预测一个OSN中的链路中产生良好的准确性，并且可以使用具有友好维护的监督方法和其他测量作为特征来进一步改进链路预测精度。

一种用于在控制复杂网络中枚举所有可能的输入节点的高效算法

地址： http://arxiv.org/abs/1703.00876

作者： Xizhe Zhang, Jianfei Han

摘要： 控制复杂网络在许多应用领域中非常重要。最近的工作发现，通过最大匹配可以获得用于控制网络的输入节点（MIS）的最小数量。然而，网络的最大匹配不是唯一的，并且可能存在许多MIS。由于计算成本高，找到所有可能的输入节点对于大规模网络是困难的。这里我们提出一种有效的算法来枚举网络的所有可能的输入节点。我们严格证明所有可能的输入节点可以通过最大匹配算法的简单修改获得，这允许我们通过仅计算一个MIS获得所有可能的输入节点（所有MIS的联合）。在合成网络和实际网络上的实验结果表明，所提出的算法的计算复杂度与前一个算法的对比明显提高。对于大规模合成网络和真实网络，观察到与以前的算法相比增加了100000x。
