一类节点复制网络模型中最短路径长度的分布

地址： http://arxiv.org/abs/1708.07177

作者： Chanania Steinbock, Ofer Biham, Eytan Katzav

摘要： 我们提出了通过节点重复（ND）演进的网络增长模型中最短路径长度（DSPL）分布的分析结果。该模型捕获了重要机制发挥重要作用的社会网络，熟人网络和科学引文网络的结构和增长动态的基本属性。从初始种子网络开始，在每个时间步骤，选择被称为母节点的随机节点进行复制。其子节点被添加到网络中，形成到母节点的链接，并且以每个邻居的概率$ p $。所得网络的程度分布符合幂律分布，因此ND网络是无标度的网络。为了计算DSPL，我们得到概率的时间演化的主方程$ P_t（L = \ ell）$，$ \ ell = 1,2，\ dots $，其中$ L $是一对节点之间的距离和$ t $是时候。找到主方程的精确解析解，我们得到$ P_t（L = \ ell）$的闭合表达式。平均距离$ \ langle L \ rangle_t $和直径$ \ Delta_t $被发现像$ \ ln t $一样扩展，即ND网络是一个小型世界网络。 DSPL的方差也可以像$ \ ln t $一样扩大。有趣的是，平均距离和直径显示出小世界网络的性质，而不是在其他无标度网络中观察到的超小型世界网络行为，其中$ \ langle L \ rangle_t \ sim \ ln \ ln t $。

在知识图中查找流以支持事实检查

地址： http://arxiv.org/abs/1708.07239

作者： Prashant Shiralkar, Alessandro Flammini, Filippo Menczer, Giovanni Luca Ciampaglia

摘要： 在线生成的信息的数量和速度限制了当前的新闻实践，以相同的速率事实检查索赔。事实检查的计算方法可能是帮助缓解大规模错误传播风险的关键。这种方法的设计不仅可以在评估可疑索赔的真实性方面具有可扩展性和有效性，而且还可以通过展示相关事实和模式来提高人事检验人员的生产力，从而帮助他们进行分析。为此，我们提出一种新颖的，无监督的基于网络流程的方法来确定以（主题，谓词，对象）三元组的形式表达的事实陈述的真实性。我们将有关现实世界实体的背景信息作为流动网络的知识图表，以及作为流体抽象商品的知识。我们表明，这样的三元组的计算事实检查相当于找到从主题节点发出的“知识流”，并通过连接它们的路径朝向对象节点流动。对娱乐，商业，体育，地理等相关事实的一系列现实世界和手工制作的数据集的评估显示，这种网络流模型可以非常有效地识别真假的虚假陈述，超越现有算法测试用例。此外，该模型具有自动发现几个有用的路径模式和表面相关事实的能力，这可能有助于人类事实检查者证实或反驳索赔。

富人俱乐部案例中核心边缘网络的弹性

地址： http://arxiv.org/abs/1708.07329

作者： Matteo Cinelli, Giovanna Ferraro, Antonio Iovanella

摘要： 核心 - 外围网络是呈现一组中心和密集连接的节点（即核心）和一组非中心和稀疏连接的节点（即周边）的结构。富人俱乐部是指最高度节点显示高密度连接的集合。因此，显示富俱乐部的网络可以解释为核心 - 外围网络，其中核心由多个集线器组成。在本文中，我们测试显示逐渐密集的富豪俱乐部的网络的弹性，并且我们观察到这种结构如何能够在信息流中的凝聚力和效率方面影响网络措施。另外，我们考虑到这样一种情况，而不是使核心更密集，我们增加了与周边环节的联系。这两个核心和边增稠程序勾勒出新环节的布局决策过程，使我们能够进行一个有助于在弹性视角下对复杂网络进行理解和监督的情景分析。两个过程的优点以及它们的含义在网络效率和节点异质性两方面进行了讨论。

测量技术复杂性 - 当前的方法和结构复杂性的新措施

地址： http://arxiv.org/abs/1708.07357

作者： Tom Broekel

摘要： 本文回顾了技术复杂度测量的两个突出的方法：反思的方法和技术组合难度的评估。它审查了他们的核心基本假设，并讨论了与这些相关的潜在问题。作为替代方案，引入了结构复杂性的新措施。本文还提出了四个具有技术复杂性的风格化事实，作为五个复杂度测量（随着时间的推移，更大的研发工作，更多的协同研发，空间集中）的实证评估的基准。评估采用1980年至2013年的欧洲专利数据，发现结构复杂性的新尺度，反映了四种风格化事实与传统措施一样好或更好。在经验应用中也显示出较少的问题。
