从安全论坛挖掘可操作的信息：恶意IP地址的情况

原文标题： Mining actionable information from security forums: the case of malicious IP addresses

地址： http://arxiv.org/abs/1804.04800

作者： Joobin Gharibshah, Tai Ching Li, Andre Castro, Konstantinos Pelechrinis, Evangelos E. Papalexakis, Michalis Faloutsos

摘要： 这项工作的目标是系统地从黑客论坛中提取信息，其信息通常被描述为非结构化：信息的文本不一定遵循任何书写规则。相比之下，许多安全倡议和商业实体正在利用现成的公开信息，但他们似乎将重点放在有组织的信息来源上。在这里，我们重点讨论在论坛中报告的IP地址中识别恶意IP地址的问题。我们开发了一种自动识别恶意IP地址的方法，其设计目标是独立于外部来源。一个关键的新颖之处在于我们使用矩阵分解方法来提取用户的行为信息的潜在特征，并将其与相关帖子的文本信息相结合。我们技术的关键设计特点是它可以很容易地应用于不同的语言论坛，因为它不需要复杂的自然语言处理方法。特别是，我们的解决方案只需要新语言中的少量关键字加上特定功能捕获的用户行为。我们还开发了一个工具来自动化安全论坛的数据收集。使用我们的工具，我们从3个不同的论坛收集大约600,000个帖子。我们的方法表现出较高的分类准确率，而在所有三个论坛中识别恶意IP的准确率均高于88％。我们认为我们的方法可以提供更多的信息：与参考黑名单VirusTotal相比，我们发现潜在恶意IP地址多达3倍。随着网络战变得越来越激烈，早期访问有用的信息对于消除黑客的先发优势变得更加迫切，我们的工作朝着这个方向迈出了坚实的一步。

社会物理学：从通信中揭示人类行为

原文标题： Social Physics: Uncovering Human Behaviour from Communication

地址： http://arxiv.org/abs/1804.04907

作者： Kunal Bhattacharya, Kimmo Kaski

摘要： 在2000年后的时代，促进人类交流的技术迅速成倍增长。虽然这些技术的采用严重影响了人们的行为和社会性，特别是在城市而且在农村地区，但他们在不同数据库上的“数字足迹”已成为一个活跃的研究领域。这种庞大的人口数据集的存在和可获得性使得科学家能够以前所未有的方式研究和模拟先天的人类趋势和社会模式，从而补充了传统研究方法，如问卷调查研究。在这次审查中，我们专注于数据分析和建模研究 - 我们称之为社会物理学 - 因为它已经使用手机数据集来深入了解人类社会性，通信突发性，移动模式和日常节奏的各个方面。

GroupSense：使用智能手机进行群组识别的轻量级框架

原文标题： GroupSense: A Lightweight Framework for Group Identification using Smartphones

地址： http://arxiv.org/abs/1804.05055

作者： Snigdha Das, Soumyajit Chatterjee, Sandip Chakraborty, Bivas Mitra

摘要： 在一个组织中，个人更倾向于组成各种正式和非正式的团体进行相互交流。因此，无所不在地识别这些群体并了解他们的动态对于监测个人的活动，行为和福祉非常重要。在本文中，我们开发了一种名为GroupSense的轻量级但接近精确的方法，通过用户的智能手机识别基于集体感应的各种互动组。来自传感器信号的组检测不是直接的，因为邻近的用户可能不总是在同一组之下。因此，我们使用从音频信号中提取的声学上下文来推断邻近对象之间的交互模式。我们已经开发出一种无监督和轻量级的用户组检测机制，通过从网络科学中提取线索，并在模块化方面测量检测到的群体的内聚性。考虑到模块性，GroupSense可以有效地消除不正确的组，并根据特定场景中接近度和声学环境所起的作用调整机制。所提出的方法已经在学术机构环境中的许多实际情况下得到实施和测试，并且我们观察到GroupSense即使在嘈杂的环境中也能够以接近90％的准确度识别用户群。
